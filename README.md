#ArrowTracker

ArrowTracker is a modern web application that tracks cryptocurrency prices in real-time. Built with vanilla JavaScript and featuring stunning visual effects, it provides instant updates on Bitcoin and Solana prices with 24-hour change indicators.

<img width="1640" height="664" alt="arrw" src="https://github.com/user-attachments/assets/4af5b0cb-8a87-4f81-a06a-1b7e3eabcc66" />

## Features

|**Real-time Price Tracking** | Live cryptocurrency prices updated automatically every 30 seconds|
| **24-Hour Change Indicators** | Visual arrows (▲/▼) showing price movement trends|
| **Pretty good looking UI** | Glassmorphic design with animated gradients and smooth transitions|
| **Responsive Design** | Works seamlessly on desktop, tablet, and mobile devices (probably)|
| **Manual Refresh** | Instant price updates with a single click|
| **Auto-updates** | Set-and-forget functionality with automatic refresh intervals|

---

##Technologies Used

- HTML5 - Semantic markup structure
- CSS3 - Advanced animations, gradients, and glassmorphism effects
- JavaScript (ES6+) - Async/await for API calls and dynamic updates
- CoinGecko API - Reliable cryptocurrency price data
- Vercel - Hosting the live demo
